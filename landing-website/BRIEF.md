# CelestCal Landing Page Build

## Goal
Build a clean, responsive landing page for CelestCal — a cycle-synced calorie tracker for women.

## Tech Stack
- HTML5
- Tailwind CSS (CDN for quick setup)
- Vanilla JavaScript (smooth scroll, fade-in animations)
- Single-page design (no backend needed yet)

## Design System

### Colors
- **Warm Ivory:** #F5F3EF (background)
- **Soft Terracotta:** #D4876F (primary CTA, accents)
- **Warm Beige:** #E8DDD3 (secondary backgrounds)
- **Soft Charcoal:** #3A3A3A (text)
- **Sage Green:** #A8B5A1 (success states)
- **Warm Coral:** #E8A698 (highlights)
- **Deep Plum:** #8B6F7D (cycle indicators)

### Typography
- **Font:** DM Sans (Google Fonts)
- **Headings:** Semi-bold (600)
- **Body:** Regular (400)
- **Line height:** 1.6

### Style
- Rounded corners: 8px
- Generous spacing (never cramped)
- Clean, modern, warm (not clinical)
- Golden hour aesthetic (warm, authentic)

## Page Structure

1. **Hero** (above fold)
   - Headline: "Finally, a tracker that gets hungrier when you do."
   - Subhead: "CelestCal adjusts your calorie target to your menstrual cycle. Because your metabolism isn't static."
   - CTA: Download on App Store badge
   - Secondary CTA: "Join Android Waitlist"
   - Visual: iPhone mockup (placeholder for now)

2. **Problem** (validate frustration)
   - Headline: "You're not failing. Your body is changing."
   - Body: Explain luteal phase hunger, metabolism variance
   - Visual: Split screen illustration (placeholder)

3. **Wedge** (cycle-synced TDEE)
   - Headline: "The only tracker that adapts to your cycle."
   - 3-step flow: Track cycle → We adjust → Stay consistent
   - Visual: Moon phase icons + TDEE graph (placeholder)

4. **How It Works** (3-step flow)
   - Snap photo → Verify → Done
   - Visual: 3-panel iPhone mockup sequence (placeholders)

5. **Features** (5 cards)
   - Best Accuracy, No Guilt, Voice Logging, Cycle Insights, Native iOS
   - Icon + headline + 1-sentence description per card

6. **Social Proof** (3 testimonials)
   - Cards with quote + name + role

7. **Pricing**
   - Free tier vs Premium comparison
   - $9.99/month or $79.99/year
   - Launch offer: First 1,000 at $59.99
   - 7-day free trial

8. **FAQ** (accordion or simple list)
   - How does cycle-synced TDEE work?
   - What if I don't track my period?
   - How accurate is AI photo logging?
   - Is this available on Android?
   - Can I cancel anytime?
   - Is my data private?

9. **Footer**
   - Repeat CTA
   - Links: About, Privacy, Terms, Support
   - Social icons (placeholder)
   - Legal disclaimer
